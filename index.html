구현 원리
SVG 파일 하나에 세 가지 <symbol>을 정의하고, CSS @media 쿼리로 표시/숨김을 제어합니다.
  • @media (max-width: 32px) → Small 심볼 표시 (viewBox 0 0 18 18)
  • @media (min-width: 33px) and (max-width: 64px) → Medium 심볼 표시 (viewBox 18 0 33 33)
  • @media (min-width: 65px) → Large 심볼 표시 (viewBox 51 0 68 68)
icon-logo-rs.svg 파일 안에 CSS와 심볼을 모두 내장했습니다. <object> 태그로 삽입하면 SVG가 독립적인 뷰포트를 갖기 때문에 내부 미디어 쿼리가 엘리먼트 자체의 width에 반응합니다.
